How to Use This Tool

Complete the tabs from left to right. Start with Setup, then add your Sales, Salaries, Expenses, and Startup Costs. The tool generates default financial projections automatically. In Outputs, you can edit month or year values before exporting. Use Export Project File to save the full submission and Import Project File to load and continue later.

Plan Setup

Set the overall projection type, start month, tax rate, opening cash, and funding assumptions. These values drive the full model.

How Sales Inputs Work

Add each product or service line separately. Enter the selling price, estimated units sold per month, and cost per unit. The tool uses this to calculate revenue, cost of sales, and gross profit.

How Salary Inputs Work
